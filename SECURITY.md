# Безопасность

## Поддерживаемая версия

Исправления безопасности применяются к текущей версии проекта в основной ветке.

## Сообщение об уязвимости

Не публикуйте сведения об уязвимости в открытом issue до её исправления.

Используйте приватный отчёт через раздел **Security → Advisories → Report a vulnerability** в репозитории GitHub. В отчёте укажите:

- затронутую страницу или инструмент;
- пошаговый сценарий воспроизведения;
- возможное влияние;
- версию браузера;
- пример файла или входных данных, если их безопасно передать.

Не прикладывайте личные изображения, секреты, токены и другие конфиденциальные данные.
